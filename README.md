# aws-iam-security-lab
Hands-on AWS IAM lab demonstrating user management, least privilege, IAM policies, groups, and MFA.

## Objective

Learn how to secure an AWS account using IAM best practices.

## Skills Demonstrated

- IAM Users
- IAM Groups
- IAM Policies
- Multi-Factor Authentication (MFA)
- Least Privilege
- AWS Console

## Tasks Completed

- Created IAM users
- Created Security and Developers groups
- Assigned permissions
- Enabled MFA
- Tested user permissions

## Lessons Learned

I learned how AWS IAM controls access to resources and why least privilege is important for cloud security.
